# 🛒 SmartCart Customer Segmentation using K-Means Clustering

## 📌 Project Overview

Customer segmentation is an important business strategy that helps companies understand different customer groups and deliver personalized marketing campaigns.

In this project, I performed **data cleaning**, **exploratory data analysis (EDA)**, **feature engineering**, and **customer segmentation** using **K-Means Clustering**. I also applied **Principal Component Analysis (PCA)** for dimensionality reduction and visualization.

---

## 🎯 Objectives

- Clean and preprocess customer data
- Perform Exploratory Data Analysis (EDA)
- Engineer meaningful features
- Identify the optimal number of clusters
- Segment customers using K-Means Clustering and Agglomerative Clustering
- Generate business insights from customer behavior

---

## 📂 Dataset

The dataset contains customer demographic information, purchasing behavior, and campaign response data.

### Features include:

- Year of Birth
- Education
- Marital Status
- Income
- Number of Children
- Customer Since
- Recency
- Wines Purchased
- Fruits Purchased
- Meat Products Purchased
- Fish Products Purchased
- Sweet Products Purchased
- Gold Products Purchased
- Web Purchases
- Catalog Purchases
- Store Purchases
- Deals Purchases
- Campaign Responses
- Complaint Status

---

## 🧹 Data Cleaning

The following preprocessing steps were performed:

- Handled missing values
- Removed duplicate records
- Converted data into appropriate formats
- Checked for inconsistent values
- Created new features:
  - Age
  - Customer Tenure
  - Total Spending
  - Total Children

---

## 📊 Exploratory Data Analysis (EDA)

Performed various visualizations to understand customer behavior:

- Correlation Heatmap
- Pair Plot
- Distribution Analysis
- Income vs Total Spending
- Feature Relationships

---

## 🤖 Machine Learning

### Feature Scaling

- StandardScaler

### Dimensionality Reduction

- Principal Component Analysis (PCA)

### Cluster Selection

- Elbow Method
- Silhouette Score

### Clustering Algorithm

- K-Means Clustering
- Agglomerative Clustering

---

## 📈 Results

The analysis successfully segmented customers into **4 distinct clusters** based on purchasing behavior and income patterns.

### Key Insights

- Customers with higher income generally spend more.
- Distinct customer groups exhibit different purchasing behaviors.
- Customer segmentation can help businesses design personalized marketing campaigns.
- PCA helped visualize customer groups effectively.
